## 📝 视频简介（直接复制到B站简介栏）

🌐 **EasyTier** 是一个开源的异地组网/内网穿透工具，功能很强，但命令行参数实在太多了……

所以我写了一个 **一键管理脚本**，全程菜单交互式操作，**不用记任何命令和参数**！

━━━━━━━━━━━━━━━━━━━━━━

🔧 **脚本功能一览**

✅ **三种模式全覆盖**
  • Web 控制台模式 — 连接公共控制台，最简单
  • 配置文件客户端模式 — 灵活配置网络名称、密钥、子网代理等
  • 服务端/中继模式 — 部署自己的中继节点

✅ **完整生命周期管理**
  • 全新安装 / 修改配置 / 更新版本 / 卸载清理
  • 查看运行日志 / 查看组网 Peer 列表 / 重启服务

✅ **模式自由切换**
  • 客户端 ↔ 服务端 ↔ Web控制台，任意切换
  • 切换不删除配置文件，随时切回来

✅ **高级配置支持**
  • 多 Peer 节点冗余（填多个服务器地址）
  • 多 CIDR 子网代理（回车结束输入）
  • 可选侦听端口（tcp/udp/ws/wss），设了自动开启中继转发
  • 加密开关、DHCP/固定IP、版本选择

✅ **贴心细节**
  • 三种下载方式：本地镜像 / GitHub代理 / 直连GitHub
  • 版本升级自动备份，失败自动回滚
  • 所有参数都有默认值，直接回车就能用
  • 彩色终端输出，状态一目了然

━━━━━━━━━━━━━━━━━━━━━━

💾 **获取方式**

```bash
curl -fsSL https://raw.githubusercontent.com/Doboo/scripts/main/easytier.sh -o easytier.sh && chmod +x easytier.sh && sudo ./easytier.sh
```

或者手动下载后执行：
```bash
sudo ./easytier.sh
```

⚠️ 需要 root 权限运行
